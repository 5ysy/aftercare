# Workflow Maps

## Review-first map

1. Detect relevant skill trigger
2. Run review
3. Classify findings
4. Decide: fix now or defer?
5. If fix now, write plan
6. Execute incrementally
7. Re-review
8. Gate merge

## Incremental refactor map

1. Stabilize tests around current behavior
2. Isolate one responsibility move
3. Apply one structural change
4. Run verification
5. Commit or checkpoint
6. Continue or stop
